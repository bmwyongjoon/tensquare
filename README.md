## 🎯 게임 목표

**10칸 보드에 수식을 완성해 100을 만들어라!**

---

## 🏆 승리 조건

3코인을 먼저 획득한 플레이어가 최종 승리!

| 결과 | 코인 |
|------|------|
| 정확히 **100** 완성 | 🪙🪙 2코인 |
| 100 이하 최고점 | 🪙 1코인 |
| 100 초과 또는 음수 | 0점 |

---

## 📦 카드 구성

모든 인원 동일하게 **64장 단일 덱** 사용

| 종류 | 구성 | 수량 |
|------|------|------|
| 숫자 카드 | 0~9 각 4장 | 40장 |
| 연산 카드 | + × − ÷ 각 6장 | 24장 |
| **합계** | | **64장** |

---

## ▶ 게임 진행

1. 연산 카드 1장씩 뽑아 선공 결정
   - × 선공 → + 2번째 → − 3번째 → ÷ 4번째
2. 각 플레이어에게 **숫자 6장 + 연산 4장 = 10장** 배분
3. 내 차례에 손패에서 카드 1장을 선택
   - **내 보드 또는 상대방 보드**에 배치 가능!
4. 손패가 모두 막히면
   - 손패 공개 → 카드 1장 버리고 새 카드 받기
5. 모든 보드 10칸이 채워지면 수식 계산 후 코인 지급

> ※ 2판부터는 전판 꼴찌가 선공
> ※ 덱 소진 시 버린 카드를 다시 셔플해 사용

## 📋 배치 규칙

| 규칙 | 설명 |
|------|------|
| ❌ | 보드 양 끝(1·10번 칸)에 연산 기호 배치 불가 |
| ✅ | 단, **−** 는 1번 칸 가능 (음수 부호) |
| ❌ | 연산 기호 연속 배치 불가 / 이중 마이너스(−−) 금지 |
| ❌ | 리딩 제로 금지 (예: 03 불가) |
| ❌ | ÷ 0 금지 (÷−0 포함) |

---

## 🔢 점수 계산 예시

사칙연산 우선순위 (× ÷ 먼저) 에 따라 계산

```
1 2 + 8 8 + 0 × 9 7  (10칸)

= 12 + 88 + (0×97)
= 12 + 88 + 0
= 100점! 🎉
```

## ✨ 주요 기능

- **PWA 지원** — 홈 화면에 앱으로 추가 가능
- **AI 난이도 3단계** — 😊 쉬움 / 🎯 보통 / 🔥 어려움
- **상대 손패 카드 수 표시** — 🃏 N장으로 실시간 확인
- **리플레이 기능** — 게임 종료 후 턴별 기록 확인 및 저장
- **진동 피드백** — 카드 배치 시 모바일 햅틱
- **사운드 효과** — 배치 / 에러 / 코인 획득음
- **BGM** — 레트로 게임 음악, 🎵/🔇 버튼으로 ON/OFF
- **게임 통계** — 인원별 기록, 순서별 승률, 100 달성 분석
