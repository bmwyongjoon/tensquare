# TEN SQUARE

> 10칸 보드에 수식을 완성해 최고 점수를 노리는 카드 전략 게임

---

## 플레이 방법

1. GitHub Pages 주소로 접속
2. 크롬 주소창 오른쪽 점 3개 → **홈 화면에 추가**
3. 앱처럼 실행

---

## 게임 규칙

- 게임 시작 시 각 플레이어에게 카드 10장 배분 (숫자 + 연산)
- 자기 순서에 손패에서 카드 1장을 **자신 또는 상대방** 보드에 배치
- 손패의 모든 카드를 놓을 수 없으면 패를 공개 → 1장 버리고 1장 교체
- 모든 보드의 10칸이 채워지면 수식 계산 후 점수 비교
- **3판 선취**한 플레이어가 최종 승자

### 인원별 손패 구성

| 인원 | 숫자 카드 | 연산 카드 |
|------|-----------|-----------|
| 2인  | 6장       | 4장       |
| 3인  | 6장       | 4장       |
| 4인  | 5장       | 5장       |

### 배치 규칙

- 보드 양 끝(1번·10번 칸)에 연산 기호 배치 불가 (단, − 는 1번 칸 가능)
- 연산 기호 연속 배치 불가 / 이중 마이너스(−−) 금지
- 리딩 제로 금지 (예: 03, −03 불가)
- 0으로 나누기 금지 (÷0, ÷−0 포함)
