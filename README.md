## 🃏 카드 구성

모든 인원 동일하게 **64장 단일 덱** 사용

| 종류 | 구성 | 수량 |
|------|------|------|
| 숫자 카드 | 0~9 각 4장 | 40장 |
| 연산 카드 | + × − ÷ 각 6장 | 24장 |
| **합계** | | **64장** |

## 🎮 게임 모드

### 💯 클래식
- 100 만들기 모드로 3코인 선취한 플레이어 승리
- 100 이하 최고점 → **1코인** 획득
- 정확히 100 달성 → **2코인** 획득 (역전 찬스!)
- 100 초과 또는 음수 → **0점**
- 동점자 발생 시 연장전 단판 진행

### 🏅 올림픽
- 5가지 종목을 한 판씩 순서대로 진행
- 모든 종목 완료 후 코인 합산 최다 보유자 승리
- 동점 시 5종목 중 랜덤 1개 선택해 연장전 단판 진행

### 🎯 선택
- 5가지 종목 중 원하는 1개를 골라 3코인 선취 방식으로 진행
- 동점 시 연장전 단판 진행

---

## 🏆 5가지 종목

| 종목 | 설명 | 코인 |
|------|------|------|
| 💯 클래식 | 100 이하 최고점 만들기. 100 정확히 달성 시 보너스 | 1코인 (100 달성 시 2코인) |
| 📈 가장 큰 수 | 제한 없이 가장 큰 숫자 만들기 | 1코인 |
| 🎲 랜덤 | 1·2번째 플레이어가 덱에서 뽑은 두 자리 목표 숫자에 가장 근접한 플레이어 승리 | 1코인 |
| 0️⃣ 제로 | 0 이상의 숫자 중 0에 가장 가까운 플레이어 승리. 음수는 0점 | 1코인 |
| 🤝 동일패 | 모든 플레이어가 0~5 숫자 6장 + 연산자 4장 동일한 손패로 시작. 가장 큰 수 만든 플레이어 승리 | 1코인 |

## ▶ 게임 진행

1. 연산 카드 1장씩 뽑아 선공 결정 (× 선공, + 2번째, − 3번째, ÷ 4번째)
2. 게임 시작 시 각 플레이어에게 **숫자 6장 + 연산 4장 = 10장** 배분
3. 내 차례에 손패에서 카드 1장 선택
4. 선택한 카드를 **자신 또는 상대방** 보드의 빈 칸에 배치
5. 손패의 모든 카드를 어디에도 놓을 수 없으면
   - 손패를 모두에게 공개 → 버릴 카드 1장 선택 → 새 카드 1장 받기
6. 모든 보드의 10칸이 채워지면 수식 계산 후 점수 비교
7. 덱 소진 시 버린 카드를 다시 셔플해 사용
8. 2판부터는 전판 꼴찌가 선공 (역순 진행)

## 📋 배치 규칙

| 규칙 | 설명 |
|------|------|
| ❌ | 보드 양 끝(1번·10번 칸)에 연산 기호 배치 불가 |
| ✅ | 단, **−** 는 1번 칸 배치 가능 (음수 부호) |
| ❌ | 연산 기호 연속 배치 불가 |
| ❌ | 이중 마이너스(−−) 금지 |
| ✅ | × − / ÷ − 음수 피연산자는 허용 |
| ❌ | 리딩 제로 금지 (예: 03, −03 불가) |
| ❌ | ÷ 0 금지 (÷−0 포함) |

---

## 🔢 점수 계산

일반 사칙연산 우선순위 (× ÷ 먼저) 에 따라 계산

```
예시: 3 + 5 × 2 − 1 + 48 ÷ 6
    = 3 + (5×2) − 1 + (48÷6)
    = 3 + 10 − 1 + 8
    = 20점
```

## ✨ 주요 기능

- **PWA 지원** — 홈 화면에 앱으로 추가 가능
- **상대 손패 카드 수 표시** — 🃏 N장으로 실시간 확인
- **리플레이 기능** — 게임 종료 후 턴별 기록 확인 및 저장
- **진동 피드백** — 카드 배치 시 모바일 햅틱
- **사운드 효과** — 배치/에러/코인 획득음
- **BGM** — 레트로 게임 음악, 🎵/🔇 버튼으로 ON/OFF
- **게임 통계** — 인원별 기록, 역대 최고 점수 TOP 5
